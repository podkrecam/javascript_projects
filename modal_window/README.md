## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

The **Modal Window Project** is a simple example of implementing modal windows in a webpage using vanilla JavaScript, HTML, and CSS. It demonstrates how to:
- Open a modal window upon clicking a button.
- Close the modal via the close button, the `Escape` key, or by clicking the overlay.
